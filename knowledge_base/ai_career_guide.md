# AI Career Guide Knowledge Base

## Artificial Intelligence

Artificial Intelligence is a field of computer science focused on building systems that can perform tasks that usually require human intelligence. These tasks include reasoning, learning, decision-making, understanding language, recognizing images, and solving problems.

## Machine Learning

Machine Learning is a branch of AI where systems learn patterns from data instead of being programmed with fixed rules. Common machine learning tasks include classification, regression, clustering, and recommendation.

## Deep Learning

Deep Learning is a subfield of machine learning that uses neural networks with many layers. It is commonly used in image recognition, speech recognition, natural language processing, and generative AI.

## Generative AI

Generative AI focuses on creating new content such as text, images, code, audio, or video. Large language models are a common type of generative AI used for chatbots, summarization, question answering, and content generation.

## Large Language Models

Large Language Models are AI models trained on large amounts of text. They can understand prompts and generate human-like responses. Examples of LLM use cases include chatbots, writing assistants, code assistants, and document question-answering systems.

## Prompt Engineering

Prompt engineering means writing clear instructions for an AI model to get better responses. A good prompt includes the task, context, expected output, constraints, and examples when needed.

## Chatbot Memory

Most LLM API calls are stateless. This means the model does not automatically remember previous messages. To create memory, the application stores previous user messages and assistant replies, then sends that conversation history with each new request.

## RAG

Retrieval-Augmented Generation, or RAG, is a technique where the application retrieves relevant information from documents or a knowledge base and sends it to the model with the user question. This helps the model answer using external information without training the model again.

## Fine-Tuning

Fine-tuning means training an existing model on a custom dataset to improve its behavior for a specific task. It is more complex than prompting or RAG and usually requires training examples, evaluation, and more resources.

## AI Engineer Roadmap

A beginner AI engineer should learn Python, NumPy, pandas, data preprocessing, machine learning basics, model evaluation, deep learning basics, NLP, APIs, prompt engineering, GitHub, and deployment tools such as Streamlit or FastAPI.

## Good AI Internship Projects

Strong beginner AI internship projects include OCR text recognition, sentiment analysis, resume screening, AI chatbot, text summarizer, career path recommender, image classifier, and document question-answering chatbot.

## Writing AI Projects on a CV

A strong AI project bullet should mention the problem, tools, model or technique, and result. It should avoid vague wording. Good words include developed, implemented, integrated, evaluated, optimized, deployed, and automated.
